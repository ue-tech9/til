# TIL

> **Today I Learned**
>
> **TIL とは「Today I Learned（今日学んだこと）」の略です。**
> 日々いろいろな言語や技術について学んだ、小さな発見を短くまとめたコレクションです。
> どれもフルのブログ記事にするほどではないけれど、メモしておきたいことたち。
>
> 公開しながら学ぶ（Learning In Public™）ことや、実務での優秀な仲間たちと協力する中で得た知見をまとめています。

**Mission: Give First.**
I share these not just for myself, but to save someone else facing the same error logs.
(自己学習や実務で得た「些細な気づき」や「トラブル解決ログ」を、未来の自分と世界のために共有します)

### ⚠️ Security & Compliance

**Strict adherence to enterprise security standards.**
(本リポジトリは、エンタープライズセキュリティ基準を厳守しています)

* **No Business Logic:** Only Infrastructure/Middleware configurations. No application logic included.
    (インフラ設定のみを含み、業務ロジックは含みません)
* **Credential Safety:** No real secrets are stored. All IDs and IPs are sanitized.
    (実在する機密情報は保存しません。IDやIPアドレスは一般化されています)
